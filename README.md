# FitFoodApi

## Description
FitFood Api is a Rest Api that allows you to track your food, calories and macros. 

## Technologies
- Node.js
- Express

## Setup
1. Clone the repository
2. Run 'node index.js' in the terminal
3. Open Postman and test the endpoints

## Endpoints
- GET api/food
- POST api/food
- PUT api/food/:id
- DELETE api/food/:id
